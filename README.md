# Nexus ITAM

Enterprise IT Asset Management application powered by Notion Database.

## Setup

1.  **Install Dependencies**
    ```bash
    npm install
    ```

2.  **Environment Variables**
    Create a `.env` file in the root directory (copied from `.env.example`):
    ```bash
    VITE_NOTION_KEY=your_secret_key
    VITE_NOTION_DATABASE_ID=your_database_id
    ```

3.  **Run Locally**
    ```bash
    npm run dev
    ```

## Field Mode
Switch to field mode for mobile-optimized asset auditing.
